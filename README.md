# mts_python_final_homework

## Основное
Итоговая домашняя работа по Python от МТС ШАД

Данный проект содержит реализацию эндпоинтов для работы с книгами (что было сделано на занятиях) и для работы с продавцами. Также реализована авторизация пользователей по JWT-токену для защиты следующих эндпоинтов: 
- **GET** api/v1/seller/seller_id
- **POST** api/v1/books/
- **PUT** api/v1/books/book_id

Все эндпоинты для продавцов покрыты тестами.

## Запуск 
`requirements.txt` содержит все зависимости, необходимые для запуска проекта.

Запуск проекта производится следующим образом:
1. Поднимается БД в докер-контейнере при помощи команды `docker-compose up -d --build`
2. Из корня директории запускается проект при помощи команды `uvicorn src.main:app`

## Замечания

В `src/tests/conftest.py` создание движка для работы с БД происходит с использованием NullPool (без его использования тесты отказывались работать). Возможно, это как-то может повлиять на работоспособность тестов
```
async_test_engine = create_async_engine(
    settings.database_test_url,
    echo=True,
    poolclass=NullPool
)
```
